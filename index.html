Les Mots à la Pelle

Les Mots à La Pelle sont nés de 4 volontés individuelles désirant œuvrer de manière collective dans le champ de la langue et de l’écriture.

Download this project as a .zip file Download this project as a tar.gz file

Déploiement

Pour déployer l'application, veuillez suivre ces indications:

$ cd path/to/www
$ git clone https://github.com/guillaume-sainthillier/malp.git

Installation

Adaptez le fichier dyn/database.inc.php à la configuration de votre serveur.
Importez le fichier mots_a_la_pelle.sql dans votre base de données.

Dépendances Serveur

Le module mod_rewrite.so doit être actif dans la configuration d'apache.
L'extension php_socket doit être disponible.
Le port 9013 doit également être ouvert et disponible.

Déploiement du WebSocket

Pour lancer le WebSocket, ça se passe par la ligne de commande: 

$ path/to/php php path/to/site/socket/serveur.php
