Malp

Les Mots à La Pelle sont nés de 4 volontés individuelles désirant œuvrer de manière collective dans le champ de la langue et de l’écriture.

Download this project as a .zip file Download this project as a tar.gz file

Bienvenue sur le site des Mots à la Pelle.

Pour déployer l'application, veuillez suivre ces indications:

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout gh-pages

Installation

Adaptez le fichier dyn/database.inc.php à votre configuration serveur.

Dépendances Serveur

Pour faire fonctionner l'application, le module mod_rewrite.so doit être actif dans la configuration d'apache L'extension php_socket doit être disponible Le port 9013 doit également être ouvert et disponible

Websocket

Pour lancer le Websocket, ça se passe par la CLI: 

$ path/to/php php path/to/site/socket/serveur.php
